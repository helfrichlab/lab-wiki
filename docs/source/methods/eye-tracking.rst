Converting edf to asc
~~~~~~~~

1. Register with SR Research.
2. Download EyeLink Data Viewer.
3. Use EDF2ASC utility.

See more `here <https://www.sr-research.com/support/thread-7674.html>`_. You need to create an account to have access to their programs.
